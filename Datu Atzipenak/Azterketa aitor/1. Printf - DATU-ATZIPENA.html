1. Printf


printfren sintaxia: printf (formatuaren Stringa, Object … datuak);


Formatuaren Stringa karaktere kate bat da eta bertan bi atal ezberdintzen ditugu:

  • testu finkoa, dagoen moduan agertuko dena.
  • zehaztapenak, % aurretik itxatxita dutenak.

datuak formatua aplikatuko zaien datuak. Datu kopurua aldagarria izango da.



Formaturen egitura


%[datuen_kokapena$][formatuaren_adierazlea][zabalera][.zehaztapena]bihurketa_karakterea


Kortxeteak, aukera adierazten dute.


BIHURKETA KARAKTEREAK

Karak.

Esanahia

d

Hamartar oinarriko zenbakia.

f

Puntu finkodun zenbaki erreala (float).

E,e

Zenbaki erreala notazio zientifikoan.

g

Zenbaki erreala oso handia edo oso txikia denean, notazio zientifikoan idatziko da.

x, X

Zenbaki osoa hamaseitar eran idatziko da.

s, S

Stringa. s:letra  xehetan S: letra larrietan(maiuskulatan)

c,C

“Unicode” karakterea. C: larrietan



FORMATUAREN ADIERAZLEAK (Flag)

Adieraz.

Esanahia

-

Lerrokatu ezkerraldera.

(

Zenbaki negatiboak parentesi artean agertuko dira.

,

Hamartar banatzailea.

+

Zenbaki positiboetan + ikurra agertuko da.

0

Zero zenbakiz bete.

You can build a formatted String and assign it to a variable using the static format method in the String class. The use of a format string and argument list is identical to its use in the printf method. The format method returns a reference to a String.
